1. Clone the [demo-apps](https://github.com/getcord/demo-apps) repository.

2. Navigate to the dashboard directory:

```bash
cd demo-apps/dashboard
```

3. Install the dependencies:

```bash
npm i
```

4. Run the app:

```bash
npm run dev
```
